# ДЗ1 команды XRUST_BEZE 

### Лента

1. Создание поста
- Вводимый текст отображается без искажений
- При нажатии на "Enter" происходит перевод строки в тексте
- Клик вне поля ввода останавливает ввод текста
- Если в поле ввода не введен текст, отображается плейсхолдер "Что нового?"
- <b>BUG: лишнее пустое пространство внутри поля ввода</b>
<img width="656" alt="баг" src="https://github.com/user-attachments/assets/a155ad86-ee9a-4db9-915c-b73de6bf1d8e">
- Выбранное вложение прикрепляется к посту
- Можно добавить несколько вложений поочередно
- Можно выбрать и добавить несколько вложений сразу
- При нажатии на крестик вложение удаляется
-  <b>BUG: вложение сначала сдвинуто вниз на 3 строки, а потом прижимается к тексту и равномерно смещается вниз</b>
<img width="656" alt="баг" src="https://github.com/user-attachments/assets/12b5738c-bf14-4c66-a2a9-8cb04df74f82">
<img width="656" alt="баг" src="https://github.com/user-attachments/assets/732ea191-6a28-4658-98df-126029527ee2">
- Можно опубликовать пост без текста, содержащий вложение
- Нельзя опубликовать {пустой пост / пост, содержащий только пробелы}
- При публикации поста {пустые строки / строки, состоящие только из пробелов} удаляются в случае, если они стоят в конце
- При вводе текста, который очищается санитайзером, публикуется пустой пост
<img width="656" alt="алерт" src="https://github.com/user-attachments/assets/7cac5d79-8a61-46eb-b783-f65ebaf58782">

- Как только количество введенных символов доходит до лимита, ввод становится невозможным
- Если опубликовать длинный пост, он сворачивается и показывается кнопка "Показать еще"
- При нажатии на кнопку "Показать еще", показывается весь пост целиком
- Чтобы свернуть развернутый вручную пост, нужно перезагрузить страницу
- К посту можно прикрепить не только картинку, но и другие файлы
- Клик по аватарке в поле создания поста переводит на профиль пользователя

3. Лайк
